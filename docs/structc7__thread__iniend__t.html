Detailed Description

C7スレッドの生成時と開放時のコールバック。

スレッドの target 関数が走る前や、スレッドが終了する段階でコールバックする 関数を指定するための構造体。最初に C7スレッドを起動するよりも前に、 c7_thread_register_iniend() で登録しておくのが望ましい。
