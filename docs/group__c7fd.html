ファイルに対するアドバイザリロックの設定/解除。

Parameters
fdロック操作対象のファイルの記述子。
enableC7_FALSE ならロックを解除し、そうでないなら有効にする。
Returns

Note
アドバイザリロックの対象領域はファイル全体となっている。
